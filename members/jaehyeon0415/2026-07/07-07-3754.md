## [3754. Concatenate Non-Zero Digits and Multiply by Sum I](https://leetcode.com/problems/concatenate-non-zero-digits-and-multiply-by-sum-i/description/?submissionId=2063524849)

### 접근 방법
- 0을 제거하면서 각 자릿수를 더하기 위해선 각 자릿수를 확인하는게 핵심이라 생각함
- n을 10씩 나눔
- `sum`: 10으로 나눈 나머지 값을 합산함
- `x`: 0이 아닌 10으로 나눈 나머지 값을 `digit`을 곱해서 더함

### 코드
```kotlin
class Solution {
    fun sumAndMultiply(n: Int): Long {
        var sum: Long = 0
        var x: Long = 0
        var digit: Long = 1

        var number = n
        while (number > 0 ) {
            val digitNum = number % 10
            if (digitNum != 0) {
                sum += digitNum
                x += digitNum * digit
                digit *= 10
            }

            number /= 10
        }

        return sum * x
    }
}
```

### 복잡도
- 시간복잡도: O(log n) - 자릿수만큼 탐색
- 공간복잡도: O(1)
