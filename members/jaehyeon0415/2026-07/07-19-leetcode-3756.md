## [3756. Concatenate Non-Zero Digits and Multiply by Sum II](https://leetcode.com/problems/concatenate-non-zero-digits-and-multiply-by-sum-ii/description/?envType=daily-question&envId=2026-07-08)

### 접근 방법
- 각 쿼리마다 직접 순회하면 오래걸림
- 큰 수를 다룰땐 전처리 작업이 필요함
- 0을 제외한 연결값, 개수, 자릿수 합을 전처리함

### 코드
```dart
class Solution {
  final int mod = 1000000007;

  List<int> sumAndMultiply(String s, List<List<int>> queries) {
    // 모든 자릿수의 합
    final List<int> prefixSum = List<int>.filled(s.length + 1, 0);

    // 0을 제외한 숫자들을 연결한 값
    // 숫자가 커질 수 있어서 mod를 적용해서 저장
    final List<int> prefixValue = List<int>.filled(s.length + 1, 0);

    // 0이 아닌 숫자의 개수
    final List<int> prefixNonZeroDigitCount = List<int>.filled(s.length + 1, 0);

    // 10^i % mod
    final List<int> pow10 = List<int>.filled(s.length + 1, 1);

    for (int i = 0; i < s.length; i++) {
      final int digit = s.codeUnitAt(i) - 48;
      final bool isZero = digit == 0;

      prefixSum[i + 1] = prefixSum[i] + digit;
      prefixValue[i + 1] = isZero 
          ? prefixValue[i] 
          : (prefixValue[i] * 10 + digit) % mod;
      prefixNonZeroDigitCount[i + 1] = prefixNonZeroDigitCount[i] + (digit == 0 ? 0 : 1);
      pow10[i + 1] = (pow10[i] * 10) % mod;
    }

    final List<int> answer = [];

    for (final query in queries) {
      final int left = query[0], right = query[1];

      final int count = prefixNonZeroDigitCount[right + 1] - prefixNonZeroDigitCount[left];
      final int digitSum = prefixSum[right + 1] - prefixSum[left];
      final int x = (prefixValue[right + 1] - (prefixValue[left] * pow10[count]) % mod + mod) % mod;

      // x * 자릿수 합을 계산
      answer.add((x * digitSum) % mod);
    }

    return answer;
  }
}
```

### 복잡도
- 시간복잡도: O(n + q)
- 공간복잡도: O(n)
