## [3658. GCD of Odd and Even Sums](https://leetcode.com/problems/gcd-of-odd-and-even-sums/)

### 접근 방법

- 첫 n개 홀수의 합 $1 + 3 + \cdots + (2n-1) = n^2$
- 첫 n개 짝수의 합 $2 + 4 + \cdots + 2n = n(n+1)$
- $GCD(n^2,\ n(n+1)) = n \times GCD(n,\ n+1)$ 인데, 연속한 두 정수는 항상 서로소이므로 $GCD(n, n+1) = 1$
- 따라서 답은 그냥 `n`
  - 직접 몇 번 손으로 써보면 감이옴
    
### 코드

```kotlin
class Solution {
    fun gcdOfOddEvenSums(n: Int): Int = n
}
```

### 복잡도

- 시간복잡도: O(1)
- 공간복잡도: O(1)
