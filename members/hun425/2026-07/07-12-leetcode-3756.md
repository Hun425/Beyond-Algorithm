## [3756. Concatenate Non-Zero Digits and Multiply by Sum II](https://leetcode.com/problems/concatenate-non-zero-digits-and-multiply-by-sum-ii/)

### 접근 방법

- 3754(I)의 쿼리 버전. 쿼리마다 구간을 직접 돌면 O(N×Q)라 시간 초과 
- 프리픽스 배열 4개를 문자열 한 번 순회로 생성
  - `cnt[i]`: 앞 i글자 중 0 아닌 숫자 개수 
  - `sum[i]`: 앞 i글자 자릿수 합 
  - `pre[i]`: 앞 i글자의 0 아닌 숫자를 이어붙인 값 (mod p)
  - `pow10[i]`: 10^i mod p 
- 핵심 공식: $x = pre[r+1] - pre[l] \times 10^k$ (단, $k = cnt[r+1] - cnt[l]$)
  - 이어붙이기는 뒤에 숫자가 붙을 때마다 앞 값이 10배씩 커지므로, 앞머리를 뺄 때는 밀린 만큼(10^k) 곱해서 빼야 함
  - 1234에서 뒤 두 자리만 남기려면 12가 아니라 1200(=12×10²)을 빼는 것과 같은 원리
- 답은 `x × (sum[r+1] − sum[l]) mod p`

### 코드

```kotlin
class Solution {
    fun sumAndMultiply(s: String, queries: Array<IntArray>): IntArray {
        val p = 1_000_000_007L
        val n = s.length

        val cnt = IntArray(n+1)
        val sum = LongArray(n+1)
        val pre = LongArray(n+1)
        val pow10 = LongArray(n+1)

        pow10[0] = 1

        for (i in 0 until n){
            val d = s[i] - '0'
            cnt[i+1] = cnt[i] + if (d == 0) 0 else 1
            sum[i+1] = sum[i] + d
            pre[i+1] = if(d!=0) (pre[i]* 10 +d) % p else pre[i]
            pow10[i+1] = pow10[i] * 10 % p
        }

        return IntArray(queries.size) { i ->
            val (l, r) = queries[i]
            val x = (pre[r+1] - pre[l] * pow10[cnt[r+1]-cnt[l]] % p + p) % p
            val sum = (sum[r+1] - sum[l]) % p
            (x * (sum % p)%p).toInt()
        }
    }
}
```

### 복잡도

- 시간복잡도: O(N + Q) — 전처리 한 번 순회 + 쿼리당 O(1)
- 공간복잡도: O(N) — 프리픽스 배열 4개

### 회고

- 생각없이 막짜다가 시간 초과 나서 문제 다시읽고 재설계 들어감
- 시간을 줄이기 위해 부분합, 자릿수 합, 이어붙인 값, 10의 거듭제곱을 미리 계산해두고 쿼리마다 O(1)로 답을 구하도록 설계
