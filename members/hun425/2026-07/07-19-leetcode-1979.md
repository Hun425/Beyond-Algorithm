## [1979. Find Greatest Common Divisor of Array](https://leetcode.com/problems/find-greatest-common-divisor-of-array/)

### 접근 방법

- 배열 전체의 GCD가 아니라 최솟값과 최댓값 딱 두 수의 GCD만 구하면 되는 문제
- 한 번 순회로 min, max를 찾고 유클리드 호제법 적용
- $GCD(a, b) = GCD(b, a \bmod b)$ b가 0이 되면 그때의 a가 답
  
### 코드

```kotlin
class Solution {
    fun findGCD(nums: IntArray): Int {
        var a = nums.max()
        var b = nums.min()
        while (b != 0) {
            val t = a % b
            a = b
            b = t
        }
        return a
    }
}
```

### 복잡도

- 시간복잡도: O(n + log(max))
- 공간복잡도: O(1)
