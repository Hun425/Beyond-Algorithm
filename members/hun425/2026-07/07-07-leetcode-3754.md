## [3754. Concatenate Non-Zero Digits and Multiply by Sum I](https://leetcode.com/problems/concatenate-non-zero-digits-and-multiply-by-sum-i/)

### 접근 방법

- n을 문자열로 바꾼 뒤 `filter`로 '0'을 전부 제거하면, 0이 아닌 숫자들이 원래 순서대로 이어붙은 문자열이 남음
- 이 문자열을 그대로 `toLong()` 하면 문제에서 요구하는 x가 되고, 각 문자를 `digitToInt()`로 더하면 자릿수 합 sum이 됨
- 답은 `x * sum`. 0이 아닌 숫자가 하나도 없으면 x = 0이므로 0을 바로 반환
- 처음에는 `split("0")`으로 접근했는데, 연속된 0이나 끝자리 0에서 빈 문자열이 생겨 `toLong()`이 터지는 문제 발견 후 `filter`로 바꾸어 해결

### 코드

```kotlin
class Solution {
    fun sumAndMultiply(n: Int): Long {
        val digits = n.toString().filter { it != '0' }
        if (digits.isEmpty()) return 0L
        val sum = digits.sumOf { it.digitToInt() }
        val x = digits.toLong()
        return sum * x
    }
}
```

### 복잡도

- 시간복잡도: O(log n) — 자릿수만큼 순회
- 공간복잡도: O(log n) — 필터링된 자릿수 문자열

### 회고

- 문제를 "0을 없애고 나머지를 더한다"로 잘못 이해해서 처음 풀이가 틀림 (영어 해석 이슈)
- `for`문을 변수에 대입하려다 (`val answer = for (...)`) Kotlin에서 `for`는 표현식이 아니라 문(statement)이라 값을 반환하지 않는다는 걸 배움
